# Shield Scanner

Shield Scanner is a Python-based ethical hacking toolkit for learning penetration testing and web reconnaissance.

## Features
- Port Scanner
- Directory Finder
- XSS Testing
- Banner Grabber
- Subdomain Finder

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/shield-scanner.git
cd shield-scanner
pip install -r requirements.txt
python shield_scanner.py
